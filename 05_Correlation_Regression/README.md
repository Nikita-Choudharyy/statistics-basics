# Module: Correlation and Regression Analysis

Welcome to the Correlation and Regression section! This module focuses on understanding how variables interact with one another and how we can use those mathematical relationships to make future predictions—forming the absolute bedrock of Machine Learning modeling.

## 🗂️ Module Contents & Sub-topics

1. **`correlation.md` (Relationship Identification)**
   - Understanding the direction and strength of relationships.
   - Deep dive into **Pearson's Correlation Coefficient ($r$)**.
   - Breaking the ultimate data myth: *Correlation vs. Causation*.

2. **`simple_linear_regression.md` (Predictive Modeling)**
   - Moving from relationship identification to prediction using $Y = \beta_0 + \beta_1X + \epsilon$.
   - Understanding the **Ordinary Least Squares (OLS)** optimization method.
   - Evaluating model performance using **R-Squared ($R^2$)**.
